How to use Dithering post effect:
1) Add Dithering.cs to Main Camera
2) Set dithering type and steps in the Dithering script (suggestion: set a power of two)

How to create Dithering Texture:
1) Press the Open Wizard button in the Dithering script or press right button in project and select Create>Dithering Texture
2) Set levels(dithering size) and path in the Wizard 
3) Press the Create.
4) Set new texture in the Dithering script.
